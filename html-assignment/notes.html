img tag
This tag allows you to add an image into your webpage

p tag
The paragraph tag allows you to add a paragraph/body of text onto your webpage

h1 tag
This is a heading 1 tag, which is the biggest heading in HTML.

ul tag
This tag adds an unordered list to your webpage, which is a list with bullet points

strong tag
This tag makes text between the opening and closing tag appear bold on the webpage
